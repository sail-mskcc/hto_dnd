# Installation

## Requirements

- Python 3.8 or higher
- pip package manager

## Install from PyPI

```bash
pip install hto-dnd
```

## Install from Source

```bash
git clone https://github.com/sail-mskcc/hto_dnd.git
cd hto_dnd
pip install -e .
```

## Verify Installation

```bash
# Check CLI is available
hto --help

# Check Python import
python -c "import hto; print('HTO-DND installed successfully')"
```
